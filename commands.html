// Reference

Linux Commands

Kumpulan command yang sering gw pake di Termux sehari-hari.

// Navigasi

pwd

Lihat posisi direktori sekarang

ls

Lihat isi folder

cd nama-folder

Masuk ke folder

cd ..

Balik ke folder sebelumnya

cd ~

Balik ke home directory

// File & Folder

mkdir nama

Bikin folder baru

touch nama.txt

Bikin file baru

rm nama.txt

Hapus file

rm -rf nama-folder

Hapus folder beserta isinya

cp file1 file2

Copy file

mv file1 file2

Pindah atau rename file

// Package Manager

pkg install nama

Install package

pkg uninstall nama

Hapus package

pkg update && pkg upgrade

Update semua package

pkg list-installed

Lihat semua package yang terinstall
