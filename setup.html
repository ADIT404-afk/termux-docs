Setup Termux

Langkah-langkah setup Termux dari nol sampai siap dipakai.

01

Install Termux

Download Termux dari F-Droid, bukan dari Play Store - versi Play Store udah outdate dan gak di update lagi.

Download F-Droid →
02

Setup Storage

Kasih izin akses storage biar termux bisa baca dan tulis file di hp lo.

termux-setup-storage
03

Update & Upgrade

Update semua package ke versi terbaru sebelum mulai install apapun.

pkg update && pkg upgrade
04

Install Package Penting

Install tools dasar yang sering di pake.

pkg install git python nodejs nano vim
