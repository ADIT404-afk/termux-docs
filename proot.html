Linux Proot

Cara install dan jalanin distro Linux di hp pake proot-distro.

01

Install Proot-Distro

Install package proot-distro dulu di Termux.

pkg install proot-distro
02

Install Distro Linux

Pilih distro yang mau lo install — Debian paling ringan dan stabil buat pemula.

proot-distro install debian
03

Login ke Distro

Masuk ke environment Linux yang udah lo install.

proot-distro login debian
04

Update Distro

Setelah login, update package list dan upgrade semua package.

apt update && apt upgrade
05

Keluar dari Distro

Kalau mau balik ke Termux biasa.

exit
